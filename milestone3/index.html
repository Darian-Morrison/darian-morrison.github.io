Intro | Specs/Tools

For our high-fidelity prototype, we built a HTML5 based website that simulates what our app would do. The website is built to be run on a mobile. The buttons on screen when clicked navigate the user through different web pages designed to look like the sections of the finished product.

This is a horizontal prototype where the user can operate the prototype however the functions are often locked to a single choice, e.g. having only one location to get direction to. The AR is not functional on the prototype with a GIF as a crude replacement. The "Back" buttons may also not link properly to your last page as this application does not have a Backend that would make this functionality simple.
