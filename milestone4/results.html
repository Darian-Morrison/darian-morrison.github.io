Augmented Reality Campus Navigation

Project Phase 4

Results Summary

On average it took the user between 30 seconds and 1 minutes to find the specified room using the application. According to survey results the buttons are easy to read, the homepage is easy to navigate, the map is easy to navigate, the map page is not distracting, and the floorplan is helpful from all access points. Changes that need to be made are making the building description more helpful, and less intrusive; making movement from one function to another more simplified; improving the other page navigations; and supplying options for colouration.

Recommendation

Glaring errors that currently exist are that the floorplans are not clearly navigable, we need to add button functionality in order to more quickly access the different floors of buildings. The buttons on each subpage need to be made larger, or in a different shape to better showcase their locality, and usage. We think, and the survey results agree, that adding an arrow to the GPS map showing which direction the user is facing, adding floorplan zoom, and additional options for customization would be beneficial.
