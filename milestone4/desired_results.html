Augmented Reality Campus Navigation

Project Phase 4

Desired Results

Quantitative results we would like collected include:

  1. How many button clicks on average did the user make?
  2. How long did it take the user to find a specific room on the map?

Qualitative questions we expect to be answered include:

  1. How easy is the application to use?
  2. Is the application efficient to use?
  3. Is there anything distracting about the application?
  4. Is it helpful?
