Augmented Reality Campus Navigation

Project Phase 4

Evaluation Plan

From our evaluation, we wish to find out whether or not the application is easy to use and efficient to perform an assigned task as well as the satisfaction and enjoyment of the users while using the application. The evaluation will take place in the the lab, which only take 20 minutes to evaluate the users. There are maximum of 15 possible users in the lab. In the beginning of the evaluation session, we will explain about the goals of the application and ask the users to sign the consent form before they begin to use the application. For the task, we would like to evaluate whether or not the application is easy to navigate and if there are anything the user would like to change, add on or prefer to remove from the application. The data that we also would like to collect is a total time that the user perform to complete the task and a total number of steps that the user use to reach ECS 660. This evaluation is in a form of usability studies as we first welcome the users, introduce the application and get consent, then observe the users as they perform task as well as follow up with the questionnaire at the end of the session. All the data collected will be interpreted in a percentage form for the quantitative questions and in a long answer form for the qualitative questions.
