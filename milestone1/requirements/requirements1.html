Building Identification

Description: The user taps on the building and the name of the building should be displayed, including any previous names of the building.

Rationale: User arrives at the unknown building and would like to check the name of the building to see if it is the right building

Fit Criterion: When the user is at the building and would like to know the name of the building

Scenario

Conor is too new to know where the David Turpin Building is for his math class. Conor decided to open the application to check where his building is. By tapping on the map, building names are shown. Conor also has the option to search the building name. Conor can use either methods to find the David Turpin Building, finding the David Turpin Building also give Conor its old name, short name, and any new name that UVic may add or change the buildingâ€™s name to. Conor finds that this application works similar to any map application but gives more information on the UVic campus.
