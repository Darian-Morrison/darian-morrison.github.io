Augmented Reality Campus Navigation

Your own clairvoyance spell

Path Mapping

Description: The user would like to know the paths and how long it will take for the user to get to another the building where the next class takes place.

Rationale: The user can set the current building location and the new building location so that the application can find and show the best 3 routes that the user can take as well as giving an approximate time the user will take to arrive at the new building. When the user chooses which path he/she will be shown the way to a classroom, the highlighted path will appear on the normal map so that the user can follow it to the building. However, if the user would like to use the AR mode to go to a classroom, there will a highlighted path showing on the screen as well.

Fit Criterion: When the user would like to know how long it will take him/her to another building and which path is the shortest path the user can take so that the user can plan ahead and arrive to class on time.

Scenario

Darian has the application opened on the phone, the application shows Darian’s location on the map and the building’s location that Darian wants to go to. Darian sees the option to Navigate to the building Darian has chosen. On the map Darian can see the paths to the building highlighted. In AR mode, Darian can see the way to the building by a path that can be seen in the AR camera, Darian finds this function to be similar to the spell Clairvoyance in the game Skyrim. Darian can turn on the voice navigation as well so that he knows for sure if he should turn right or left.
