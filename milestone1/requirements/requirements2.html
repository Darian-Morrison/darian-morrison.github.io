Interior Mapping

Description: When the user arrives at the building for the first time, the user can choose to use the building’s floor map to help finding the classroom.

Rationale: When the user is at the building and on the classroom floor, the user can follow the GPS from the user’s current location to the classroom.

Fit Criterion: After user arrives at the building, the user would like to find the classroom

Scenario

Jessica would like to find out how to navigate through the building that her classroom is in. Jessica is unfamiliar with the building because Jessica has never been in this building before. Jessica checks the application and finds that there is the ability to view the building’s floor plan. The floor plan shows Jessica room numbers, the corridor and the staircases within the building. Jessica notice that there are also options for Jessica to tap on to change the floor of the floor plan. Jessica was able to find her class in the building with the application.
