Augmented Reality Campus Navigation

Your own clairvoyance spell

Mandate

In the main map, if the user taps on a building, and the name of the building pops up – including any previous names the building may have been known as. Next, the user taps a small button that came up with the building name that shows the building’s floor plan; now the user can not only find the building more easily, but identify where in the building they need to be.

In this application, a user can click a button, hold their device’s camera up to the building, and the application identifies what building they’re looking at. This augmented reality addition could be very helpful when a traditional map leaves the user a little confused. From this building identification, the user can, once again, pull up the floor plan to navigate the building.

There are often many sidewalks, and roads leading throughout campus. Is there a better way to get from one place to another? Are there shortcuts that are known only to campus veterans? Again, this application can help with that. Type in a building’s name, previous name, abbreviation, or nickname, and choose from the option to navigate to or from said building, from your current location, or to or from another building. With the AR function, a path illuminates on the user's screen to visually show the route. Additionally, the application predicts how long the walk might be between these two points, allowing for the addition of the option to arrive at a certain time.
