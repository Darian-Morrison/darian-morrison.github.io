Your own clairvoyance spell

AR Navigation

Description: The user can use the AR mode on the application recognize the building. After recognizing the building, the application will show the name of the building to the user

Rationale: Instead of using a map with GPS on the application, the user would like to use AR mode to find the name of the building. The user can switch to the AR mode and use the device’s camera to identify the building. The user fits a portion of the building in front of them into the frame and the application uses a certain number of points to recognize the building and display the name of the building on the screen. From here the user has a number of options they can choose from like displaying the building’s floor plan, or use the building as a destination for navigation.

Fit Criterion: When the user doesn’t want to use the normal campus map and would like to use AR mode to explore the campus.

Scenario

Heather doesn’t like the the map function, or the GPS is malfunctioning, Heather decides to try the AR mode instead of the regular map mode. The AR can recognize the building in front of Heather, giving Heather the building’s name and the choice to view the building’s floor plan. This is the building that Heather is looking for!
