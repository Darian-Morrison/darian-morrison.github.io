Exterior Mapping

Description: The user follows the GPS on the map in order to find the building.

Rationale: Instead of using an AR mode on the application, the user prefers to use the map with GPS to find the building. The GPS can be use in the building as well when the user chooses to use the floor map as the background map to find the classroom

Fit Criterion: When the user doesn’t want to use the AR mode and would like to use a map to explore the campus.

Scenario

Étienne is a guest lecturer on campus and wants to find the way to ECS. Étienne has been to UVic before but ECS is a building Étienne haven’t been to since he usually visits Bob Wright building. Although Étienne is reluctant about using the application, his friend from UVic insists he tries this application to navigate his way towards ECS. Étienne opens the application to see the map function of the application. Étienne is able to locate where he is on the map. Étienne can now navigate through the map and find where ECS is on the map thanks to his friend from UVic’s brief description that ECS is southwest of Bob Wright building. Since the map application clearly shows where North is, Étienne simply choose the buildings South West of Bob Wright one by one until he sees the name of the building ECS.
