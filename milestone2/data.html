Data

In order to finalize the application we will need to collect the following data:

  • Floorplans for all the buildings on, and off campus associated with the University of Victoria (this includes the residential buildings, and child care center that are just outside of Ring Road.): these should be easily requested from facilities management.
  • SDK of Vuforia (https://www.vuforia.com/engine.html ): a powerful AR development tool offering “dynamic object recognition”
  • Using Vuforia, a database of buildings will need to be built for the AR building recognition aspect of the application
  • Mapping will be done by leveraging Google Maps’ API.
  • Shortest path data: using the mapping data, integration of short cuts, and campus knowledge to provide directions between two points.
